Qué es HTML?

HTML es el esqueleto de una página web. No hace que se vea bonito, solo organiza y estructura todo. Referencia: https://www.arsys.es/blog/que-es-html-guia-de-desarrollo-web

Head and Body

HeAd: tiene la información que necesita el navegador, por ejemplo, el título de la página web. BODY: Es todo el contenido que se podrá ver en la página web. ReFeReNcIa: https://www.arsys.es/blog/que-es-html-guia-de-desarrollo-web

CSS

Qué es CSS?

CSS es para hcer la página web bonita o fea. Referencia: https://lenguajecss.com/

Purpose of CSS

El propósito de CSS es definir un estilo de diseño. Mientras HTML nos da la "estructura", CSS muestra cómo se va a visualizar ese contenido. Referencia: https://www.w3schools.com/css/css_intro.asp

How to apply CSS?

Inline: use the style attribute inside HTML elements.
Internal: style element in the head section.
External: link element to link to an external CSS file.
Referencia: https://www.w3schools.com/css/css_intro.asp

Selector and CSS rule

Selector: es para buscar el elemento que quieres cambiar. CSS rule: son los estilos que tiene tu html. Referencia: https://www.w3schools.com/css/css_selectors.asp

Basic CSS

Tag selector: ubica elementos html basados en su nombre.
Class selector: seleccionar y style todos los elementos que tengan un mismo atributo.
Tag + class: ubicar un tipo de elemento específico dentro de una clase en específico.
Id selector: seleccionar un único elemento por el id del atributo.
Referencia: https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Styling_basics/Basic_selectors

Class VS ID

Class: elementos que comparten características. ID: es un identificador ÚNICO. Referencia: https://www.reddit.com/r/learnprogramming/comments/11rgz7q/id_vs_class_in_css/

CSS Box Model

Box that wraps around every html element. Four parts: cpntent, padding, borders, and margins. Referencia: https://www.w3schools.com/css/css_boxmodel.asp

Media queries

Media queries permite aplicar estilos CSS según el tipo de dispositivo, como resolución de pantalla o el ancho del viewport del navegador. Referencia: https://developer.mozilla.org/es/docs/Web/CSS/Guides/Media_queries/Using

Layout CSS

GRID

Permite dividir una página en áreas o regiones principales (tamaño, posición y capas). Referencia: https://developer.mozilla.org/es/docs/Web/CSS/Guides/Grid_layout

FLEX

Se utiliza para un diseño unidimensional con filas o columnas. Referencia: https://www.w3schools.com/css/css3_flexbox.asp
